# SGBD Lab1 - Database Management App

A simple Windows Forms application built with .NET 8.0 for managing a SQL Server database. This app allows users to connect to a database, view data, and add new records to a table.

---

## Features

- **Connect to Database**: Easily connect to a SQL Server database.
- **View Data**: Display data from the `Lant` table in a grid.
- **Add Records**: Insert new records into the `Lant` table using a form.

---

## How It Works

1. **Database Connection**: The app uses a connection string to connect to your SQL Server database.
2. **Data Display**: Click the "Connect" button to load and display data from the `Lant` table.
3. **Data Insertion**: Enter data into the text boxes and click "Add" to insert new records into the table.

---

## Getting Started

1. Clone the repository:
2. Open the project in Visual Studio or your preferred IDE.
3. Update the connection string in Form1.cs to match your SQL Server setup.
4. Build and run the application.
